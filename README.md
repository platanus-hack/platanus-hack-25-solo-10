## Cómo iniciar la aplicación Rails

1. Instala las dependencias:
   ```bash
   bundle install
   npm install
   ```

2. Configura la base de datos:
   ```bash
   rails db:setup
   ```

3. Inicia el servidor:
   ```bash
   bin/rails server
   ```
4. Inicia los jobs
```bash
   bin/jobs
   ```

5. Inivia Vite
```bash
   bin/vite dev
   ```

Accede a la aplicación en [http://localhost:3000](http://localhost:3000)
